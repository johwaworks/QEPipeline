# QEPipeline

VFX Project Management System

## Features

- Project Management
- Shot Management
- Worker Assignment
- Partner System
- Real-time User Status

## Tech Stack

### Frontend
- HTML, CSS, JavaScript
- Static files

### Backend
- Python Flask
- MongoDB

## Setup

### Frontend
1. Navigate to `frontend` directory
2. Open `index.html` in a browser
   - Or run: `python -m http.server 8000`

### Backend
1. Navigate to `backend` directory
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure MongoDB connection
4. Run server:
   ```bash
   python app.py
   ```

## Deployment

- Frontend: AWS Amplify
- Backend: AWS Lightsail / ngrok
